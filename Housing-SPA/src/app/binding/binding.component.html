4. Two way Data Binding

We can use two way binding to bind any property defined in Typescript file using square bracket and paranthes inside it and and using ngModel inside that as directive i.e [(ngMode)]="yourname". In that way you do not need to do anything at component level and property automatically be updated on the change of the element under which it is defined
